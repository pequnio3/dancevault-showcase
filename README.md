# DanceVault Showcase

Source for the public showcase site of [DanceVault](https://github.com/pequnio3) — a Flutter app for capturing, cataloging, and practicing dance moves, by Blake Carpenter.

**Live site:** https://pequnio3.github.io/dancevault-showcase/

The DanceVault application source is private during active development. This repo only contains the marketing site.

## Updating

Plain HTML/CSS/JS, no build step. Edit `index.html` / `styles.css` / `script.js`, commit, push. GitHub Pages auto-deploys from `main`.
